1. Скачать файлы в отдельную папку
2. Запустить файл main.py
3. Если требуется визуализация парсинга проверить/установить драйвера для Microsoft Edge (выбран, так как на всех ОС Windows пристуствует, начиная с ОС windows 8), для MAC os выбран Safari (не тестилось)
4. Если из файла requirements.txt не установились зависимости, из файла install скопировать строку и вставить в терминал PyCharm. Дождаться загрузки и установки зависимостей.
5. Повторно запустить файл main.py.
